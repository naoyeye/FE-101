FE 101 是面向全体豌豆（包括非攻城狮）的一个前端入门系列 talk，主要目标是普及前端开发的基础知识。

随着公司规模的不断扩大，各种地方都会有前端开发的需求，FE 101 旨在解决前端开发供需不平衡的状况：在需要 Web 前端开发的产品中，让更多人能够自己完成不需要专职前端工程师支持的工作，此其一。其二，布道，让更多人了解 Web 前端到底是做什么，是怎么做的，使我们能和其他人更好的配合工作。

整个课程体系分为由浅入深的三个阶段，每个阶段都包含数个 topics，每个 topic 会是一到两课时不等的时间，每个人都可以根据自己的基础和兴趣点灵活按需参加。

HTML 基础

讲师：Junyu
课程时间：2014-03-04 13:30 - 14:30

课程内容：HTML 是前端最基础和核心的技术之一，是所有信息在浏览器中的实际载体。HTML 是一种看起来很简单，却一眼就能通过其看出一个前端工程师基础知识是否扎实的技术。本节介绍 HTML 相关的基础知识，包括元素、属性，常用标签等，以及内容、结构和表现相分离、语义化等基本概念。

课前准备
Wikipedia: HTML
W3C HTML Educational material for beginners.

讲师 ziming
课程时间：2014-03-11 13:30 - 14:30

课程内容：「人靠衣装马靠鞍」，前端开发强调内容、结构和表现分离。HTML 是「结构」，而 CSS 就是「表现」了，她决定了一个页面最终如何呈现在用户眼前。本节会介绍 CSS 的基础语法、选择器、权重、常用样式等基础知识，以及 layout 的原则和简单技巧。

DOM 模型和 jQuery API 简介

讲师：yaochun
课程时间：2014-04-08 13:30 - 14:30

课程内容：和其它 UI 技术不同，前端的界面开发没有 Canvas 或 Stage 这样的概念（HTML5 Canvas 除外），在我们将一幅幅高大上的设计稿变成可交互的界面的过程中，大多数情况下是操作一个叫做 DOM 的对象。「跨平台」是前端开发最重要的优势之一，但由于不同浏览器厂商实现的 DOM API（其它 API 也有这种问题） 存在差异，导致前端开发从跨平台跳出来，却跳进了跨浏览器的泥坑，jQuery 最初就是为了解决这个问题而出现的，同时也封装了其它很实用的 API，让前端工程师能更专注于业务本身。本节将介绍 DOM 和 jQuery API 相关的内容。
